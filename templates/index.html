SaaS Project Management API

A multi-tenant REST API for managing projects and tasks across organizations, with role-based access control and async reporting.

API Documentation OpenAPI Schema

Multi-Tenant Isolation

Every organization's data is fully isolated. Querysets are scoped at the viewset level — cross-tenant access returns 404, not forbidden.

Role-Based Access Control

Three roles: Owner, Admin, Member. Owners and Admins can delete projects; Members have read/write access to tasks without destructive operations.

Async Reporting

Request a project report via POST; a Celery worker aggregates task counts and completion percentage asynchronously. Poll for results via GET.

JWT Authentication

Register an organization, receive access and refresh tokens. Stateless authentication with configurable token lifetimes.
